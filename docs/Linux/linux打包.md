# Linux 把源代码打包成安装包


## source code to deb

正常编译安装源代码的需要如下操作：

```
./configure
make
make install
make clean
```

那么如果仅仅是要编译成安装包呢？

先去安装checkinstall

> apt-get install checkinstall

这里以的debian为例使用checkinstall打包了一个deb的安装包

```
tar -zxvf source-app.tar.gz
cd source
./configure
make
checkinstall
```

然后使用dpkg -i name.deb即可
